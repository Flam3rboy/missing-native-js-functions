setIntervalNow​

▸ setIntervalNow(callback, milliseconds?, ...args): number | NodeJS.Timeout

Repeatedly calls a function or executes a code snippet, with a fixed time delay between each call, immediately executes the first callback on setup, after that waits for the given delay

Example

setInterval((a, b) => {
 console.log(a, b)
}, 500, "param 1", "param 2"); // prints param 1 param 2 every 500 ms

See

https://developer.mozilla.org/en-US/docs/Web/API/setInterval

Parameters​

NameTypeDescription
callbackFunctiona function to be executed every delay milliseconds. The first execution happens after delay milliseconds.
milliseconds?numberThe time, in milliseconds (thousandths of a second), the timer should delay in between executions of the specified function or code.
...argsany[]Additional arguments which are passed through to the function specified by func once the timer expires. *
