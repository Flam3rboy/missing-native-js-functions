merge​

▸ merge(obj): any

Merge the original object with the given object. Keys will be overwritten if they occure in both objects, the last occurrence will be kept in the resulting object

Example

{a: 1, b: 2, c: 3}.merge({d: 4, e: 5, f: 6}) // {a:1, b:2, c: 3, d: 4, e: 5, f: 6}
{a: 1, b: 2, c: 3}.merge({a: 4}) // {a:4, b:2, c: 3}
